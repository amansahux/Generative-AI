2️⃣ Machine Learning (ML)

Human Defination 

Machine ko itna capable banana ki wo khud se seekh sake. use bs data dedo wo khud rules bana lega like if u give 1000 cats and 1000 dogs then yr automatically difference pata kr legga 

Technical Definition 

Machine Learning is a subset of AI that focuses on the development of algorithms that allow computers to learn from data and make predictions or decisions without being explicitly programmed.

📚 Chapter 2: NLP (Natural Language Processing)
Human Definition

Natural Language Processing (NLP) ek AI field hai jo computer ko human language ko read, understand, analyze aur generate karna sikhati hai.

Simple words me:
NLP = Human Language + Computer

Technical Definition

NLP is a branch of Artificial Intelligence that enables computers to process, understand, interpret and generate natural human language.

Natural Language ka matlab?

📚 Chapter 3: LLM (Large Language Models)
Bhai, ye pura GenAI ka heart hai. ❤️

Human Definition

LLM ek bahut bada AI model hai jo billions of text examples padhkar language ke patterns seekhta hai aur next token predict karke text generate karta hai.
Technical Definition

A Large Language Model is a deep neural network trained on massive text datasets to predict the next token in a sequence.
